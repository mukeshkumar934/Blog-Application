# Blog Application

A React blog application built using Vite and json-server.

## Features
- Blog list and detail view
- Mock API using json-server
- Component-based architecture
- Responsive layout

## Tech Stack
- React
- Vite
- JavaScript
- json-server

## How to Run
1. Start API:
   json-server --watch src/db.json --port 3000

2. Start React app:
   cd frontend
   npm install
   npm run dev
